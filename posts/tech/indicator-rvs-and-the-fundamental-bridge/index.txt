13:Tc2bf,<p><em>Expectation</em> (also called <em>expected value</em> or <em>mean</em>) is a very important concept in probability and statistics. One usage we could name is <em>estimation</em> in statistics in which we try to infer the parameters of a statistical model. Another simple application is the step of <a href="http://cs231n.github.io/neural-networks-3/#sanitycheck">sanity checks</a> in training a nerual network: verify if the loss function gives a value which is acceptably around its expectation.</p>
<p>There are a few ways to compute the expectation of a <em>random variable</em>. Apart from <a href="https://en.wikipedia.org/wiki/Law_of_the_unconscious_statistician">LOTUS</a>, we could obtain it using the <em><strong>fundamental bridge</strong></em>. This post will introduce <em>indicator random variables</em> (r.v.s); and the bridge between this kind of r.v.s and its expectation.</p>
<h3 id="indicator-rvs">Indicator r.v.s</h3>
<p>An <em><strong>indicator random variable</strong></em> for an event <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>A</mi></mrow><annotation encoding="application/x-tex">A</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.6833em;"></span><span class="mord mathnormal">A</span></span></span></span> is a random variable which takes only 2 values: 1 (when <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>A</mi></mrow><annotation encoding="application/x-tex">A</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.6833em;"></span><span class="mord mathnormal">A</span></span></span></span> occurs) and 0 (otherwise). We usually denote it as <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>I</mi><mi mathvariant="normal">_</mi><mi>A</mi></mrow><annotation encoding="application/x-tex">I\_A</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.9933em;vertical-align:-0.31em;"></span><span class="mord mathnormal" style="margin-right:0.0785em;">I</span><span class="mord" style="margin-right:0.0278em;">_</span><span class="mord mathnormal">A</span></span></span></span>.</p>
<p>It seems trivial, right? Yet, it provides a powerful connection between probability and expectation. This link is called <em>fundamental bridge</em>.</p>

<h3 id="discussion">Discussion</h3>
<p>When I first read the definition of indicator r.v.s, I thought <em>"So trivial! Is it really worth a mention"</em>. But it turns out to be a quite useful tool to calculate expectations.</p>
<p>Conventionally, determining expectations could be done using LOTUS. Sometimes, the analytical transforms required could be troublesome whereas using the <em>fundamental bridge</em> may result in a very elegant solution.</p>
